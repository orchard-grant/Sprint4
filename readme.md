# Web App Grant Orchard

This web app follows a Django tutorial to complete an extensive web app. 


## Instructions for Build and Use

Steps to build and/or run the software:

1. Download VS Code, and all needed extensions
2. Download Django, and set up a virtual environmnet.
3. Create another folder for site specific app.

Instructions for using the software:

1. Run the server in the terminal of VSCode
2. Click on the link for the app in the server
3. Navigate the app, log in, add questions, and play around with it.

## Development Environment 

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* VS Code
* Django
* Python
* a web browser to open the app
